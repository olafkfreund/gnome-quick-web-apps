Turn any website into
a first-class GNOME app

A native GTK4 & libadwaita web-app manager. Paste a URL — it detects the PWA manifest, grabs the icon, and installs a launcher into your app grid. Each app runs isolated, with its own window and dock identity.

Features

Built for the modern desktop

🔎

PWA manifest autofill

Paste a URL and the name, icon, theme colour and navigation scope fill themselves in from the site's Web App Manifest.

🎨

Automatic icons

Pulls the highest-resolution manifest or apple-touch icon, with a clean lettered fallback when a site has none.

🧭

Scope confinement

Real PWA behaviour: links that leave the app's scope open in your system browser instead of hijacking the window.

🪟

Native window identity

Each app gets its own icon in the dock and Alt-Tab via StartupWMClass, installed through the XDG launcher portal.

🛡️

Per-app controls

Custom user agent, ad-blocking content filters, zoom and injected CSS — configured per web app.

🎬

Full site compatibility

Bundled Chromium (CEF) rendering means DRM and Chrome-only sites just work.
